# Netflix Data Analysis Project

## Project Overview

This project analyzes Netflix data using Python, Pandas, and Matplotlib.

## Tools Used

- Python
- Pandas
- Matplotlib
- Git
- GitHub
- VS Code

## Dataset Used

- netflix_titles.csv

## Analysis Performed

1. Top 10 Countries on Netflix
2. Movies vs TV Shows
3. Release Year Trend
4. Top 10 Ratings

- README.md
- top10_countries.png
- movies_vs_tv.png
- release_year_trend.png
- top_ratings.png
